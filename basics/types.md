# 数据类型

## 基础类型

### 数值类型
- int: 整数类型
- float: 浮点数类型
- byte: 字节类型

### 字符串
- string: 字符串类型
    - 支持单引号和双引号
    - 支持多行字符串

### 布尔值
- bool: true/false

### 空值
- null: 表示空值

## 复合类型

### 数组
    arr = [1, 2, 3]
    arr = ["a", "b", "c"]

### 字典/映射
    dict = {"name": "yak", "version": "1.0"}

### 函数类型
    fn callback(x) {
        return x * 2
    }

## 类型转换

- toString(): 转换为字符串
- toInt(): 转换为整数
- toFloat(): 转换为浮点数

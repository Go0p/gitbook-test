# Yak 语法规则

## 基本语法

### 变量声明与赋值

变量声明不需要关键字，直接使用赋值符号：

示例：
    name = "yaklang"
    age = 25
    isEnabled = true

### 函数定义

使用 fn 关键字定义函数：

示例：
    fn add(a, b) {
        return a + b
    }

### 注释语法

支持单行和多行注释：

示例：
    // 这是单行注释
    
    /* 这是多行注释
       可以包含多行内容
    */

## 运算符

- 算术运算符: +, -, *, /, %
- 比较运算符: ==, !=, >, <, >=, <=
- 逻辑运算符: &&, ||, !

## 代码块

使用花括号定义代码块：

示例：
    if x > 0 {
        // 代码块内容
    }

## 命名规范

- 变量名：使用小驼峰命名法
- 函数名：使用小驼峰命名法
- 常量：使用大写字母和下划线

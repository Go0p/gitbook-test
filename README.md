# 简介

Yak 是一个为安全研究人员设计的嵌入式脚本语言。它提供了强大的安全测试能力和友好的语法特性。

## 特性

* 简单易学的语法
* 丰富的安全测试功能
* 完善的标准库支持
* 高性能的执行效率
* 良好的可扩展性
